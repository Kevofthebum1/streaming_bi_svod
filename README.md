# Streaming Platform Strategy Analysis 

## Project Overview 

This project analyzes the content strategy of major streaming platforms using publicly available datasets.

## Objective 

Identify strategic opportunities for a selected streaming platform. 

## Dataset 

Movies and TV Shows on Streaming Platforms dataset. 

## Methodology 

* Data Cleaning
* Exploratory Data Analysis
* Platform Comparison
* Strategic Recommendation

## Key Insights

* Hulu content generally maintains strong overall quality
* Content is heavily concentrated in Reality TV and Drama
* Limited diversity in genre offerings
* Opportunity to expand into underrepresented genres
